# omikuji_app

おみくじアプリです。  
「おみくじを引く」ボタンを押すと、画面が遷移して  
「大吉」「中吉」「吉」「小吉」「末吉」「凶」(画像は中吉と吉のみ)  
のどれかが、ランダムに表示されます。
